Theme Installation

To install a theme on your Bruce Firmware device:

  • Download the theme ZIP file
  • Put the files on your device's SD card in any directory
  • Navigate to Settings > UI Themes
  • Select your new theme from the list

For detailed instructions, check the installation guide.
